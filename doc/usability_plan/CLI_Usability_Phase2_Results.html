#######################################################
Musings after Lori leaves:

   We should define what an experiment is in terms
   of commands. Describe a typical session roadmap.

   1) expCreate

       Initial definition of an experiment. Application
   and data model.

   2) expAttach

       Allows the user to include more attributes to a
   predefined experiment.

   3) expGo

       Experiment has been completely defined and is to
   be run.

   4) expView

       Tell tool you want to look at the results that have
   been collected.

   5) exit

       Exit openss.
